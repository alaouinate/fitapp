# FitVision Pro

FitVision Pro is a high-performance Progressive Web App (PWA) designed for fitness tracking and AI-powered nutrition analysis. It uses a "Clean Architecture" inspired design implemented in Vanilla JS/CSS/HTML for maximum portability and performance.

## Design System
- **Theme**: Ultra-Dark (#0a0e27) with Electric Green accents (#00ff88).
- **Font**: Inter (Google Fonts).

## Features
1.  **Dashboard**: Real-time calorie tracking and weight trend analysis (via Chart.js).
2.  **Workout**: Dynamic daily workout plans with equipment toggles.
3.  **AI Meal Scanner**: Simulated AI Vision API to analyze meal photos for macros.
4.  **Responsive**: Hybrid layout (Sidebar on Desktop, Bottom Nav on Mobile).

## Running the App
To run the application locally, start a simple Python HTTP server:

```bash
python -m http.server 8000
```
Then visit: `http://localhost:8000`
